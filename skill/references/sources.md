## 搜索技巧

1. **限定时间范围**
   - 添加日期关键词：`"2026年4月"`, `"今日"`, `"本周"`

2. **排除无关结果**
   - 使用 `-` 排除：`AI芯片 -招聘 -培训`

3. **限定网站**
   - 使用 `site:`：`site:36kr.com AI`

4. **精确匹配**
   - 使用引号：`"Claude 3.5 Sonnet"`

5. **多关键词组合**
   - 使用 OR：`(GPT-4 OR Claude) 发布`
